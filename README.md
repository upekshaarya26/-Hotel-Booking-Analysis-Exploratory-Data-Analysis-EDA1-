# 🏨 Hotel Booking Analysis – Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project performs an Exploratory Data Analysis (EDA) on hotel booking data to uncover booking patterns, customer behavior, cancellation trends, and revenue-driving factors. The analysis provides actionable insights to help hotel management improve occupancy, reduce cancellations, and optimize pricing strategies.

---

## 🎯 Objective
- Analyze hotel booking behavior.
- Identify factors affecting booking cancellations.
- Understand customer segmentation and market trends.
- Discover seasonal booking patterns.
- Generate business recommendations to improve hotel performance.

---

## 📊 Dataset Information
- **Dataset:** Hotel Booking Dataset
- **Records:** 119,390
- **Features:** 32
- **Hotel Types:** City Hotel & Resort Hotel
- **Time Period:** More than 2 years

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🔄 Data Preprocessing
- Handled missing values
- Removed duplicate records
- Standardized date columns
- Created new features:
  - Stay Length
  - Total Guests
  - Year-Month
  - Guest Category (Domestic/International)

---

## 📈 Exploratory Data Analysis
The analysis includes:

- Booking Trends
- Cancellation Analysis
- Hotel Type Comparison
- Lead Time Analysis
- Customer Segmentation
- Market Segment Performance
- Monthly Booking Trends
- Arrival Day Analysis
- Revenue Analysis
- ADR (Average Daily Rate) Analysis

---

## 📌 Key Insights
- City Hotels experience higher cancellation rates than Resort Hotels.
- Longer lead times are associated with a greater probability of cancellation.
- Repeat guests are more reliable and less likely to cancel bookings.
- Weekend arrivals (Friday & Saturday) have the highest cancellation rates.
- OTA bookings contribute significantly to cancellations.
- August records the highest booking volume and cancellations.
- International guests generally generate higher booking value.

---

## 💡 Business Recommendations
- Introduce stricter cancellation policies for City Hotels.
- Encourage repeat bookings through loyalty programs.
- Promote advance confirmation for long lead-time bookings.
- Improve policies for non-refundable bookings.
- Plan staffing and pricing strategies around peak booking seasons.
- Offer personalized services for international guests.
